Murder of Crows + BOIDS in Unity DOTS

2 weeks | Unity, C#
A game where you control a flock of crows in a 3D environment, using BOIDS algorithm. Later made a dataoriented implementation in Unity DOTS.

Technologies used:

  • Unity (DOTS)
  • C#
  • Data oriented programming
  • BOIDS-algorithm for bird movement
  • Optimization with octrees

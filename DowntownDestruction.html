Downtown Destruction

A game I made with 4 other people for Lame Jam 43 and ranked fairly high, being #2 overall and #1 in the "enjoyment" category. This Game Jam was 72 hours long, but I unfortunately did not have time to work on the game during that whole period. The theme for this Game Jam was “Chaos”, and the game is an endless runner, where you are rewarded for running into objects to cause chaos. If you run into humans or boxes, your rage meter will build, and with full rage you can run into any object to earn score.

Some of my contributions:

  • First Basic iteration of Player Movement
  • Player Input Handling
  • Procedural Level Generation
  • Object pooling and obstacle spawning
  • Obstacle interactions and hitboxes

Procedural Generation

For this Jam, I got to use what I learned from my Spelunky prototype regarding Procedural Generation when I programmed the level generation for this game. Like in that prototype, I used a tag system where the ones making prefabs could tag the road sections with different tags depending on where and when that road section should appear. This time, I combined the procedural generation with Object Pooling for spawning in the buildings on the side and the obstacles in the road.
